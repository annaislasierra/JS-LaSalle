Un array és una llista de valors ordenada que es guarden en una sola variable i als quals s'accedeix mitjançant un índex numèric. Usa [].

Els arrays són objectes complexos; per això tenen la propietat lenght.

console.dir() mostra per consola un llistat amb totes les propietats i mètodes que té un element complex (array o objecte).

console.table() mostra per consola una taula amb totes les propietats i mètodes que té un element complex (array o objecte).

Un objecte és una col·lecció de parells clau-valor (propietat-valor), on els valors poden ser qualsevol tipus de dada. Usa {}.

Als valor d'un objecte no s'accedeix per cap índex, sinó directament per les seves propietats.
